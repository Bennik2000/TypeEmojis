# TypeEmojis
Tray application wich replaces :) with 😊 while typing

Run the application and start typing in your favourite chat program

 Available emojis         |           |
| ------------- |:-------------:|
| :)  | 😊 |
| :(  | 😕 |
| :P  | 😛 |
| <3  | ❤️ |
| O:) | 😇 |
| :*  | 😘 |
| \*.\* | 😍 |
| O.O | 😱 |
| :,) | 😅 |
